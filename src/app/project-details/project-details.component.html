Plateforme de Réservation d'Événements

Events Booking Platform

Une application web monopage pour découvrir des concerts et réserver des billets avec une expérience fluide, axée sur le mobile. Les utilisateurs peuvent parcourir les événements, consulter les détails, ajouter des billets à un "Vibebook" personnel (panier), ajuster les quantités et confirmer les achats. L'application inclut des routes protégées par authentification, des totaux en temps réel, des notifications de succès, et un effet de confettis festif lors de l'achat. Elle est optimisée pour les performances et déployée sur GitHub Pages avec une gestion appropriée du SPA 404.

A single-page web application to discover concerts and book tickets with a smooth, mobile-first experience. Users can browse events, view details, add tickets to a personal "Vibebook" (cart), adjust quantities, and confirm purchases. The app includes authentication-protected routes, real-time totals, success notifications, and a celebratory confetti effect on purchase. It's optimized for performance and deployed on GitHub Pages with proper SPA 404 handling.

Défis & Solutions

Challenges & Solutions

  • Budget de bundle dépassé — Bootstrap déplacé vers CDN et ressources resserrées pour réduire le bundle initial sous les seuils.
  • Dépréciation Sass — Suppression des imports dépréciés et utilisation du CSS compilé via angular.json.
  • Erreur 404 sur liens profonds GitHub Pages — Implémentation de la redirection SPA 404 (404.html + script index) pour les routes Angular.
  • Base href incorrecte sur Windows Git Bash — Assuré que base href est défini sur /events/ et évité la corruption de chemin via les flags CLI.
  • Fichiers spec compilés en prod — Exclusion de *.spec.ts dans tsconfig.app.json pour éviter que les globaux de test ne cassent les builds.
  • Bundle budget exceeded — Moved Bootstrap to CDN and tightened assets to drop the initial bundle below thresholds.
  • Sass deprecation — Removed deprecated imports and relied on compiled CSS via angular.json.
  • GitHub Pages deep-link 404 — Implemented SPA 404 redirect (404.html + index script) for Angular routes.
  • Incorrect base href on Windows Git Bash — Ensured base href is set to /events/ and avoided path mangling via CLI flags.
  • Spec files compiling in prod — Excluded *.spec.ts in tsconfig.app.json to prevent test globals from breaking builds.
